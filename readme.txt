== Description ==

You can put a user rating system for your blog. The visitor of your blog or website can rate your blog. You can see how they response with your blog. Fully Ajax implementation. Fast loading. Nice design.
Features:

* Self Improvement : You can improve your website functionalities. 
* Cute Look: Transparent plugin will give your blog a nice look
* Girls Icons:Very impressive girls icons are used for giving more attraction to the rating.

== Installation ==

1. Unzip the files 
2. Upload sexyrate directory (containing sexyrate.php file with image directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

4. Go to Design and edit , and paste the code `<?php wp_show_my_rate() ?>`  anywhere as you wish (header.php or any file)


== Frequently Asked Questions ==


= Should i edit any files? =
